# FlowMind

**UX Decision Simulator — محاكي قرار المستخدم**

اختبر فكرة مشروعك قبل أن تبدأ بالتنفيذ. محاكي ذكي يساعدك على فهم كيف قد يتصرف المستخدم داخل مشروعك، وأين قد يتردد أو يخرج قبل إتمام الطلب.

> نموذج عرض من **أفق التقنية** — يعمل بالكامل داخل المتصفح، بدون Backend، بدون تسجيل، وبدون إرسال أي بيانات.

## المميزات

- محاكاة من 9 أسئلة قصيرة موزعة على 4 شاشات
- محرك Rule-based شفاف ومكتوب باللغة العربية
- يدعم نوعين رئيسيين من المشاريع: **تطبيقات الجوال** و**مواقع الويب**
- نتيجة فورية: 4 درجات مركّبة + Timeline لرحلة المستخدم + 3 مشاكل + 3 توصيات
- نموذج تواصل مرتبط بواتساب (Click-to-Chat)
- LocalStorage لحفظ الجلسات محليًا

## التقنيات

React 19 + TypeScript + Vite 8 + Tailwind 3 + Zustand + lucide-react + framer-motion + react-router

## التشغيل المحلي

```bash
npm install
npm run dev
```

ثم افتح المتصفح على العنوان الذي يظهر في الـ terminal.

## البناء للإنتاج

```bash
npm run build
npm run preview
```
